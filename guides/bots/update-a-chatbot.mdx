## Create API Guide

The Chatbot Creation API allows you to create a new chatbot by making a POST request to the `/bots`.

## Endpoint

```
PATCH https://s5hxhnuksu6hwzu6dvnquqzw7q0nrqxe.lambda-url.us-east-1.on.aws/bots/{id}
```

## PATH PARAMETERS

<ResponseField name="id" type="string" required>
Id of the chatbot
</ResponseField>

## Request Headers

The API request must include the following headers:

- `x-api-key: <Your-Secret-Key>` - The secret key for authenticating the API request.

## Request Body

The request body should contain the following parameters:

<ResponseField name="name" type="string" required>
The name of the chatbot.
</ResponseField>
<ResponseField name="featured" type="array">
Array of featured products.
</ResponseField>
<ResponseField name="products" type="array">
Array of products.

  <Expandable title="Products">
    <ResponseField name="name" type="string">
    The name of the name.
    </ResponseField>
    <ResponseField name="description" type="string">
    The name of the name.
    </ResponseField>
    <ResponseField name="image_url" type="string<url>">
    The name of the name.
    </ResponseField>
    <ResponseField name="url" type="string<url>" required>
    The name of the name.
    </ResponseField>

  </Expandable>
</ResponseField>
<ResponseField name="email_support" type="boolean" required>
The email address for support.
</ResponseField>

<ResponseField name="image" type="string<url>">
The image of the chatbot.
</ResponseField>

<ResponseField name="modified" type="string<date-time>" default="2024-02-29T20:52:02.100166">
The date the chatbot was modified.
</ResponseField>

<ResponseField name="role" type="string" default="customer_support">
Role of the chatbot.
</ResponseField>

<ResponseField name="customize" type="object" required>

  <Expandable title="Products">
    <ResponseField name="color" type="string" required>
      The color of the bot
    </ResponseField>
    <ResponseField name="conversation_style" type="string" default="chat">
      The conversation style of the bot
    </ResponseField>
    <ResponseField name="language" type="string" default="english">
      The language of the bot
      Enum: `"french"`,`"english"`
    </ResponseField>
    <ResponseField name="company_name" type="string" required>
      The company name of the bot
    </ResponseField>
    <ResponseField name="welcome_message" type="string" required>
      The welcome message of the bot
    </ResponseField>
    <ResponseField name="schedule_demo_link" type="string" >
      The schedule demo link of the bot
    </ResponseField>
    <ResponseField name="sub_heading" type="string" required>
      The sub heading of the bot
    </ResponseField>
    <ResponseField name="offline_messages" type="boolean" default="false">
      The offline messages of the bot
    </ResponseField>
    <ResponseField name="deck_ai_branding" type="boolean" default="false">
      Deck AI branding of the bot
    </ResponseField>
    <ResponseField name="position_left" type="boolean" default="false">
    Chatbot position
    </ResponseField>
    <ResponseField name="chat_inputbox_placeholder" type="string" required>
    Chat inputbox placeholder
    </ResponseField>
  </Expandable>
</ResponseField>


<ResponseField name="is_gpt_4" type="boolean" default="false">
Is the bot GPT-4?
</ResponseField>
<ResponseField name="gpt_api_key" type="string" >
GPT API Key
</ResponseField>
<ResponseField name="human_support_email" type="string<email>" >
Human support email
</ResponseField>
